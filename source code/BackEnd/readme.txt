เปลี่ยนแปลงตัวแปรจากไฟล์แรกทั้งหมดจาก
    From -> src
    To -> dst

เปลี่ยนตัวแปรในไฟล์
    past_from -> Past_Src
    past_to -> Past_Dst


การทำงานกับ DB (ใช้ MongoDB)
    -จะทำการ Replace ข้อมูลลงไปใน DB ทุกครั้งแทน
        กรณีที่ใน collection ไม่มีข้อมูล จะสร้างข้อมูลเองใน /insert
    ** นั่นหมายความว่าใน db จะมีข้อมูลแค่ 1 ชุดเท่านั้น
**update ตัวหน้าเวปให้ได้ตามที่ต้องการจากข้อ 1 และ 2 ที่ให้มา
